## Problem Statement
The objective of this project is to build, evaluate, and deploy multiple machine
learning classification models to predict the presence of heart disease using
patient medical attributes.

## Dataset Description
The dataset used is the UCI Heart Disease dataset. It contains more than 900
patient records with 13 medical attributes such as age, cholesterol, blood
pressure, and chest pain type. The target variable indicates the presence or
absence of heart disease.

Dataset Source:
https://www.kaggle.com/datasets/redwankarimsony/heart-disease-data

## Models Used
- Logistic Regression  
- Decision Tree  
- K-Nearest Neighbors  
- Naive Bayes  
- Random Forest (Ensemble)  
- XGBoost (Ensemble)

## Evaluation Metrics
- Accuracy  
- AUC Score  
- Precision  
- Recall  
- F1 Score  
- Matthews Correlation Coefficient (MCC)

## Deployment
The application is deployed using **Streamlit Community Cloud**, providing a
publicly accessible web interface to evaluate and compare model performance.
